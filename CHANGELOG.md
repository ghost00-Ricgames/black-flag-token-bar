# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-01

### Added
- **NPC/PC Condition Toggle**: GMs can now view and toggle 14 standard conditions directly on a selected NPC/PC target.
- **Roll Requests**: Added a button to request Death saves from players via chat.
- **Auto-Pan**: Clicking a PC portrait in the bar now automatically pans the canvas to that token.

### Changed
- **UI Refinement**: Expanded the Bar for a second row for direct conditions access.
